# Taylor Swift

For the Swiftie who codes. Thirty-six status messages that pull from every era, leaning into the deep
cuts and lore (the vault, the easter eggs, folklore and evermore, Midnights, a little TTPD) rather than
just the radio hits, each twisted into the work of shipping code. Debug like it is me trying, merge like
you survived the Great War, ship it long story short.
