# Pokemon

Forty status messages that turn Claude Code's wait time into a Kanto and Johto road trip. Gym battles,
type matchups, the Safari Zone, blowing into the cartridge when it would not boot. If you grew up with a
Game Boy in your hands, this one is for you.

Pairs well with the `replace` mode so every spinner is on theme.

Maintained by [@Krythz43](https://github.com/Krythz43).
